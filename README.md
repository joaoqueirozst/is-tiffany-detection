# is-tiffany-detection
> Project developed based on a microservice architecture for detecting the hexapod robot [Tiffany](https://github.com/Penguin-Lab/tiffany) within the [Lab. SEA](https://github.com/labsea), IFES Intelligent Space ('is'), in partnership with [Lab. Penguin](https://www.eletroifes.com.br/penguin).

This project presents a Computer Vision system for detection and localization of a hexapod robot inside an Intelligent Space ('is') environment using Artificial Neural Networks and a microservices-based architecture. The service implemented was the `tiffany_detector`.

The detection system receives image streams from cameras installed in the laboratory and performs real-time inferences to identify the robot's position in the environment, indicating it through a `boundbox`.
